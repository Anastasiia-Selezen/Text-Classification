<br><i>Tools:</i> Python, Jupyter Notebook
<br>
<br><i>Libraries:</i> Scikit-learn, Pandas, Numpy, Matplotlib, Seaborn, nltk.
<br>
<br><i>Data Source:</i> Kaggle
